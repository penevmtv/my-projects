
# Form Validation

A simple form validation project built with vanilla JavaScript. It validates username, email, password, confirm password, and optional company number fields using regex patterns with red border feedback for errors and a "Valid" message on successful submission.

## Features
- Username: 3-20 characters, letters and numbers only
- Email: Valid email format with @ and domain extension
- Password: 5-15 characters, word characters only
- Confirm Password: Must match password
- Company Number: 1000-9999 (optional, shown when "Is Company?" is checked)
- Visual feedback with red border on invalid fields
- Valid message appears when all fields are correct

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Regex patterns for validation

## How to use
1. Open `index.html` in your browser 
**https://htmlpreview.github.io/?https://github.com/penevmtv/my-projects/blob/main/05-form-validation/index.html** 
2. Fill in the form fields
3. Click Submit to validate
4. Invalid fields will turn red, and a "Valid" message will appear if all fields are correct
