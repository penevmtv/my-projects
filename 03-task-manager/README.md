A simple task management application built with vanilla JavaScript.

## Features
- Add tasks with description and due date
- Move tasks through stages: Open → In Progress → Complete
- Delete tasks from any stage

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to use
1. Open index.html in your browser
2. Add a task using the form
3. Use Start/Finish buttons to move tasks
4. Delete tasks when no longer needed
