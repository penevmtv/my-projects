## Central Cinema

A movie ticket management application built with vanilla JavaScript. Users can add movies to the screen list, archive tickets by entering the number of tickets sold, and calculate total revenue per movie.

The HTML intentionally contains minimal IDs and classes to practice DOM tree traversal and manipulation, encouraging navigation through parent-child relationships rather than relying on convenient selectors.

### Features:
- Add movies with name, hall, and ticket price
- Archive movies by entering tickets sold
- Automatically calculate total revenue 
- Delete archived movies
- Clear entire archive
